## Descripcion
Plugin de terminal para generar comandos IOS de Cisco automaticamente para Packet Tracer.

## Plugins incluidos

### 1. pt_claude_pro.py - Command Center
Genera comandos rapidos para:
- Port Security
- VLANs
- SSH
- TACACS
- Wireless WRT300N

### 2. pt_topology_builder.py - Topology Builder
Construye topologias completas de una sola vez incluyendo todos los temas.

## Instalacion
```bash
git clone https://github.com/EMMANUELVELASQUEZ/cisco-pt-plugin.git
cd cisco-pt-plugin
python3 pt_claude_pro.py
```

## Uso rapido
```bash
python3 pt_claude_pro.py        # Command Center
python3 pt_topology_builder.py  # Topology Builder
```
